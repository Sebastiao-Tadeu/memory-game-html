Jogo da Memória - Imagem e Texto

Regras do Jogo:

1. O jogo pode ser jogado em equipes ou individualmente.

2. Uma equipe continua jogando enquanto acertar.

3. Se errar, passa a vez para a outra equipe.

4. Ganha quem encontrar mais pares de cartas.

Instruções do Jogo da Memória

Bem-vindo ao Jogo da Memória - Imagem e Texto!

Este jogo é uma ferramenta educativa interativa projetada para auxiliar no aprendizado de forma divertida. Você pode jogar sozinho ou em equipes, usando o conteúdo padrão (se houver) ou personalizando completamente as cartas!

Como Jogar:

  1. Preparação Inicial:
    • Clique no botão "Escolher arquivo" na tela inicial.
    • Selecione um arquivo no formato CSV (.csv) que contém os dados das cartas que você deseja usar. (Veja a seção "Personalização do Conteúdo" abaixo para aprender a criar seu próprio arquivo CSV).
    • Após o arquivo ser carregado com sucesso, o botão "Começar Jogo" aparecerá. Clique nele para iniciar!
  2. Objetivo do Jogo:
    • Encontrar todos os pares de cartas correspondentes.
    • Tente fazer isso no menor número de tentativas possível para ganhar uma medalha melhor!
  3. Durante o Jogo (Regras Básicas):
    • Clique em duas cartas para virá-las e revelar seu conteúdo.
    • Se as cartas formarem um par (tiverem o mesmo `id` no arquivo CSV), elas permanecerão viradas para cima.
    • Se não formarem um par, elas serão viradas para baixo novamente após um curto instante. Tente memorizar suas posições e conteúdos!
    • Modo de Equipes: Se estiver jogando em equipes, uma equipe continua jogando enquanto acerta os pares. Se errar, a vez passa para a outra equipe.
  4. Fim de Jogo e Vitória:
    • O jogo termina quando todos os pares forem encontrados.
    • No modo de equipes, ganha a equipe que encontrar mais pares. Se houver empate no número de pares, o critério de desempate pode ser o menor número de tentativas totais da equipe.
    • No modo individual, você vence ao encontrar todos os pares e sua performance será avaliada pelo sistema de medalhas.

Configurações do Jogo:

Antes de iniciar, clique no botão "Configurações" para ajustar o jogo às suas preferências:

  • Modo Single-Player ou de Equipes: Escolha se vai jogar sozinho ou em modo colaborativo/competitivo com duas equipes.
  • Habilitar Leitura das Cartas: Se ativado, o jogo usará a síntese de voz do seu navegador para ler o conteúdo textual das cartas ou a descrição alternativa (alt) das imagens quando elas forem viradas.
  • Dimensões das Cartas: Personalize a largura e a altura das cartas para melhor visualização ou para ajustar a dificuldade.
  • Limitar número de pares: Você pode definir quantos pares deseja jogar. Se o seu arquivo CSV tiver mais pares do que o limite definido, o jogo selecionará aleatoriamente o número de pares especificado. Digite `0` para usar todos os pares disponíveis no arquivo CSV.

Personalização do Conteúdo (Avançado)

Crie seu próprio jogo da memória com imagens e textos personalizados usando um arquivo CSV (Valores Separados por Vírgula).

Formato do Arquivo CSV:

Seu arquivo CSV deve ter a seguinte linha de cabeçalho na primeira linha, seguida pelos dados das cartas:

id,type,content,alt
  • id: Um número ou identificador único para cada par. As duas cartas que formam um par devem ter o mesmo 'id'.
  • type: Defina como image para uma carta de imagem, ou text para uma carta de texto.
  • content (Conteúdo da Carta):
    • Se type="image":
      • Para esta versão online do jogo: Você DEVE usar uma URL (link direto) de uma imagem hospedada na internet (ex: de sites como Imgur.com). O link direto geralmente começa com https://i.imgur.com/... e termina com .png, .jpg, etc. Caminhos para arquivos do seu computador (ex: C:\minhas_fotos\gato.png) não funcionarão aqui.
      • Se você baixar o jogo para jogar offline (do seu computador): Você poderá usar caminhos relativos para imagens salvas no seu computador (ex: minhas_imagens/gato.png, assumindo que a pasta `minhas_imagens` está junto do arquivo `index.html` do jogo).
    • Se type="text": Escreva o texto que você quer que apareça na carta.
  • alt (Texto Alternativo/Descrição):
    • Se type="image": Escreva uma breve descrição da imagem. Isso é muito importante para acessibilidade (para pessoas que usam leitores de tela) e também será o texto lido se a opção "Habilitar Leitura das Cartas" estiver ativa.
    • Se type="text": Este campo pode ser deixado em branco, ou você pode adicionar uma nota se desejar.

Exemplo de Linhas em um Arquivo CSV:

id,type,content,alt
1,image,https://i.imgur.com/kSaaKeo.png,Duas cargas elétricas positivas com linhas de campo se repelindo
1,text,Cargas Elétricas de Mesmo Sinal,
2,image,https://i.imgur.com/w3lkwdF.png,Duas cargas elétricas de sinais opostos com linhas de campo se atraindo
2,text,Cargas Elétricas de Sinais Opostos,
3,text,Fotossíntese,Processo vital para as plantas
3,text,Processo pelo qual as plantas produzem seu alimento usando luz solar.,

Após criar e salvar seu arquivo com a extensão .csv, use o botão "Escolher arquivo" na tela inicial do jogo para carregá-lo.

Dicas para um Bom Jogo:

  • Concentre-se! Preste muita atenção às cartas que são viradas, mesmo que não sejam as suas.
  • Comece pelos cantos ou bordas: Alguns jogadores acham mais fácil memorizar essas posições.
  • Crie associações: Tente associar o conteúdo da carta com sua posição na grade.
  • Divirta-se! O aprendizado é mais eficaz quando é divertido.
